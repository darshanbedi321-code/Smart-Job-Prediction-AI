# 🎯 Smart Job Market Analyzer

An end-to-end AI-powered career analysis tool that predicts job roles, estimates expected salary, and generates personalized career insights using Machine Learning and LLM-based explanations — trained on real-world data scraped from AmbitionBox.

**🔗 Live App:** [smart-job-prediction.streamlit.app](https://smart-job-prediction-ai.streamlit.app/)

---

## 📌 Overview

Smart Job Market Analyzer takes a user's skill set as input and:

1. **Predicts the most suitable job role** using a trained classification model
2. **Estimates expected annual salary** using a regression model
3. **Generates a human-like career insight** explaining the prediction, using an LLM
4. **Surfaces top hiring companies** for that role — with ratings, average salary, and open positions

The goal is to give users a quick, data-backed snapshot of where their current skills place them in the job market, along with actionable next steps to improve their prospects.

---

## ✨ Features

- 🔍 **Role Prediction** — Classifies user skills into the most relevant job role
- 💰 **Salary Estimation** — Predicts expected annual salary based on skill profile
- 🧠 **AI Career Insight** — Natural-language explanation of the prediction, generated by an LLM in an easy-to-understand tone
- 🏢 **Top Hiring Companies** — Ranked list of companies hiring for the predicted role, with rating, average salary, and open roles
- 🌐 **Fully Deployed** — Live, interactive web app accessible to anyone

## ⚙️ How It Works

1. **Data Scraping** — Job listings and company reviews are scraped from AmbitionBox, capturing skills, roles, salaries, and company ratings.
2. **Model Training**
   - A **Random Forest Classifier** is trained on skill combinations to predict the most likely job role (84.5% accuracy).
   - An **Gradient Boosting** is trained to predict expected salary based on role and skill features (R² = 0.64).
3. **LLM Explanation** — Once a prediction is made, the result is passed to Groq's `llama-3.3-70b-versatile` model, which generates a friendly, personalized explanation of the prediction along with improvement tips.
4. **Company Matching** — The predicted role is matched against the scraped company dataset to surface the top hiring companies, sorted by relevance and average salary.

## 🚀 Getting Started (Run Locally)

```bash
# Clone the repository
git clone https://github.com/darshanbedi321-code/smart-job-market-analyzer.git
cd smart-job-market-analyzer

# Create and activate a virtual environment
python -m venv venv
venv\Scripts\activate      # Windows
source venv/bin/activate   # macOS/Linux

# Install dependencies
pip install -r requirements.txt

# Add your API keys in a .env file
GROQ_API_KEY=your_groq_api_key_here

# Run the app
streamlit run app.py
```
